# Mini Autorizador | Desafio

### Rodando os testes
Para executar o teste é decessário subir o banco pelo comando:
```
docker-compose up -d
```
#### Após isso, executar
```
./mvnw test
```
### Para acessar o swagger da aplicação, basta subir a aplicação:
```
./mvnw spring-boot:run
```
#### E acessar
```
http://localhost:8080/swagger-ui.html
```
